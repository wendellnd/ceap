Significado de Web

O que é Web

Web é uma palavra inglesa que significa teia ou rede. O significado de web ganhou outro sentido com o aparecimento da internet.

A web passou a designar a rede que conecta computadores por todo o munto, World Wide Web(WWW). Web pode ser uma teia de aranha ou um tecido e também se utiliza para designar uma trama ou intriga.
